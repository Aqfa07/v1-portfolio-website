## 🎨 PROMPT 1 — Redesign Tampilan (Visual Overhaul)

```
Redesign my personal portfolio website with a modern, dark-themed aesthetic.
The portfolio belongs to Aqil Afif, a fresh graduate in Informatics Engineering
Education from Universitas Negeri Padang, specializing in Machine Learning,
Web Development, and Mobile Development.

Apply the following design improvements:

1. **Hero Section**: Add an animated typing effect for the tagline. Include a subtle
   particle or mesh gradient background. Add a glowing avatar ring animation.

2. **Skill Bars**: The proficiency bars currently show 0% — replace them with
   animated circular progress rings or horizontal bars that fill on scroll with
   accurate percentages (e.g., Web Dev 85%, ML 80%, Mobile 75%, DB 70%).

3. **Color Palette**: Use a dark navy (#0a0f1e) base with electric cyan (#00d4ff)
   and soft purple (#7c3aed) as accent colors. Avoid generic purple gradients.

4. **Typography**: Use "Clash Display" or "Cabinet Grotesk" for headings and
   "DM Sans" for body text. Avoid Inter or Roboto.

5. **Section Transitions**: Add smooth scroll-triggered fade-in animations for
   each section (About, Skills, Experience, Projects, Contact).

6. **Project Cards**: Redesign as glassmorphism cards with hover lift effect,
   tech tag pills, and a subtle gradient border on hover.

7. **Navigation**: Make it sticky with a frosted glass background. Add an
   active section indicator that highlights as the user scrolls.

8. **Footer**: Add a subtle animated gradient line, social links with hover
   icon animations, and copyright.

Tech stack: Next.js + Tailwind CSS + Framer Motion.
Keep all existing content (name, bio, skills, projects, experience, contact info) intact.
```

## 📋 PROMPT 2 — Konten Lebih Informatif & Lengkap

```
Improve the content quality and informativeness of my portfolio website.
Current content is about Aqil Afif — a fresh graduate in Informatics Engineering
from Universitas Negeri Padang with expertise in ML, Web Dev, and Mobile Dev.

Make the following content enhancements:

1. **Hero Section**: Rewrite the tagline to be more impactful. Add a brief
   "Open to Work" badge with a pulsing green dot. Include quick stats below
   the CTA buttons: "3+ Projects Deployed | 2 BNSP Certifications | C1 English".

2. **About Section**: Add a short personal mission statement. Separate the
   education and contact info into cleaner visual cards. Add a "Fun Facts" 
   or "What Drives Me" section with 3-4 icon+text items.

3. **Skills Section**: Add accurate proficiency percentages. Group skills
   visually (Frontend, Backend, ML/AI, Mobile, Tools). Add small tech logos/icons
   next to each skill tag.

4. **Experience Section**: Expand each role with 2-3 bullet point achievements.
   Add measurable impact where possible (e.g., "Trained 30+ students in Linux
   network administration"). Add a timeline visual connector between roles.

5. **Projects Section**: For each project, add:
   - A short 1-sentence problem statement
   - Key technical challenge solved
   - Screenshot or mockup placeholder
   - Status badge (Completed / In Progress / Open Source)

6. **Certifications Section**: Make it a dedicated visual section with
   certification logos, issuing organization, validity period, and a
   "Verify" link badge.

7. **Contact Section**: Add estimated response time (e.g., "Usually replies
   within 24 hours"). Add availability calendar or "Currently available for
   freelance/full-time" status.

Keep all factual information accurate. Do not fabricate credentials or projects.
```

## ⚡ PROMPT 3 — Performa & SEO

```
Optimize my Next.js portfolio website (aqil-afif.vercel.app) for performance,
SEO, and accessibility. The site belongs to Aqil Afif, a software developer
from Padang, Indonesia.

Apply the following technical improvements:

1. **SEO Meta Tags**: Add comprehensive Open Graph tags, Twitter Card meta,
   canonical URL, structured data (JSON-LD for Person schema), and a
   dynamic <title> with relevant keywords like "Aqil Afif | Machine Learning
   Developer Indonesia".

2. **Performance**:
   - Lazy load all images with next/image
   - Add blur placeholder for profile photo
   - Preconnect to external font sources
   - Code-split project modals

3. **Accessibility (a11y)**:
   - Add proper aria-labels to icon-only buttons (GitHub, LinkedIn links)
   - Ensure color contrast ratio meets WCAG AA (4.5:1 minimum)
   - Add focus-visible styles for keyboard navigation
   - Add alt text to all images

4. **Loading States**: Add skeleton loaders for the projects grid.

5. **404 Page**: Create a custom 404 page matching the portfolio design.

6. **robots.txt & sitemap.xml**: Generate and include these for better
   search engine indexing.

7. **Analytics**: Add Vercel Analytics or a lightweight privacy-first
   analytics snippet.
```
